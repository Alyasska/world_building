# Pages Preview

This folder is a temporary static UX inspection layer published through GitHub Pages.

## Purpose
- Quick browser review when the real local runtime is inconvenient to run.
- Mock the currently active MVP slice only.

## Rules
- Keep it aligned with the real runtime shell and active entities.
- Do not add speculative sections that the real app does not support yet.
- Prefer lightweight client-side mock behavior over growing a second application architecture.

## Current Coverage
- Dashboard
- Characters
- Places
- Tags
- Basic Character<->Place links
- Search
